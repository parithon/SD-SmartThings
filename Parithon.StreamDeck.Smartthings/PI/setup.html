Personal Access Token

Personal access tokens are used to authorize interaction with the API for non-SmartApp use cases. When creating personal access tokens, you can specifiy the permissions granted to the token. These permissions define the OAuth2 scopes for the personal access token. This plugin requires the following OAuth2 scopes:

  1. List all devices
  2. Controll all devices
  3. See all scenes
  4. Controll this scene

Please request a personal access token and supply the token below.
